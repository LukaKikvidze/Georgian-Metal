# Angular рЃЋрЃћрЃЉрЃърЃарЃЮрЃћрЃЦрЃбрЃў

рЃћрЃА рЃљрЃарЃўрЃА Angular-рЃќрЃћ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃњрЃЋрЃћрЃарЃЊрЃўрЃљрЃюрЃў рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА, рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃЋрЃўрЃЊрЃћрЃЮ рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃА.
